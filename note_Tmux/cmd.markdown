# Tmux命令使用
## 0x1.基础操作
> **前排提示:tmux里的`id`和`name`本质是一样的。**
> 
| 作用                       | 命令                 | 备注                             |
| -------------------------- | -------------------- | -------------------------------- |
| 顺序id启动一个新会话       | `tmux`               |                                  |
| 新建一个对话，自定义会话名 | `tmux new -s <name>` | `id`的本质其实就是纯数字的`name` |
